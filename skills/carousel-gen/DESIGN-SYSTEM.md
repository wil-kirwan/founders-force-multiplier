# Carousel Design System + Art Direction

Read this file before generating ANY carousel visuals. It defines the visual aesthetic and contains prompt templates for every slide type.

---

## Aesthetic: "Tech-Editorial"

The visual language sits between a Notion document, a magazine editorial, and hand-written class notes. Premium but approachable, technical but human.

### Background Treatment
- Soft warm muted tones: cream (#F5F0E8), beige (#EDE8DF), light gray (#F0F0ED), blush pink (#F5EBE8)
- NEVER pure white (#FFFFFF) or dark/black backgrounds
- Subtle paper grain texture for depth
- Slight warmth - like aged paper or a linen notebook

### Typography Feel (described in prompts, rendered by text overlay)
- **Headlines:** Elegant serif, large, expressive (Playfair Display in overlay)
- **Body text:** Clean modern sans-serif, readable (Plus Jakarta Sans in overlay)
- **Code/technical:** Monospace, slightly smaller (JetBrains Mono in overlay)
- The 3-font mix creates a "smart whiteboard" energy

### Color Philosophy
- Predominantly neutral palette: charcoal (#2C2926) text on warm backgrounds
- ONE accent color per carousel, used sparingly for highlights only
- Accent options: mint/sage green (#7FB685), coral (#E8836B), soft blue (#6B9FE8), amber (#E8B86B), lavender (#9B8FE8)
- Color highlights, it does not decorate

### Hand-Drawn Annotations (generated by Nano Banana)
- Arrows: curved, slightly wobbly, pointing to key elements
- Circles: hand-drawn, irregular, highlighting key terms
- Underlines: wavy or straight, emphasizing words
- Stars/sparkles: small, near surprising stats
- Style: looks sketched with a felt-tip pen, not computer-generated

### Recurring Visual Elements
- Rounded card containers (12px radius, subtle drop shadow) for screenshots
- Sticky note / tape callout boxes for tips and asides
- Dark code/terminal blocks (#1E1E2E) with colored dots (red/yellow/green) top-left
- Subtle progress indicator (small dots) near bottom
- Handle/watermark placement: bottom-right or bottom-center, small and unobtrusive

## Dimensions

- **Canvas:** 1080 x 1350 px (4:5 portrait, Instagram carousel standard)
- **Safe zone:** 60px padding all sides
- **Aspect ratio for Nano Banana:** 4:5

## Slide Types + Nano Banana Prompt Templates

### IMPORTANT: Visual Layer Only
All prompts below generate the VISUAL LAYER ONLY. They must include:
**"DO NOT include any text, words, letters, numbers, or written content in the image."**

Text is added separately via the Playwright text overlay. The visual layer should leave clear ZONES where text will be placed (negative space, lighter areas, text-safe regions).

---

### Type: COVER (Slide 1)

**When:** Always slide 1.

**Prompt template:**
```
Instagram carousel cover slide design, 1080x1350 portrait format. Warm cream/beige background (#F5F0E8) with subtle paper grain texture.

[VISUAL ELEMENT - one of:]
- Stylized avatar/portrait composited in lower-right quadrant [if using avatar reference]
- Product screenshot in rounded card container with subtle drop shadow, center-right [if tool-focused]
- Editorial lifestyle illustration in warm tones [if conceptual]

Hand-drawn curved arrow annotation pointing toward the center-left area (where headline text will go). Small hand-drawn stars/sparkles near top.

Leave large clear negative space in the upper-left and center for headline text overlay. Leave bottom-center clear for handle text.

Editorial magazine aesthetic, warm natural tones, paper texture, hand-crafted feel. NOT a Canva template. NOT corporate. NOT minimalist-flat.

DO NOT include any text, words, letters, numbers, or written content in the image.
```

---

### Type: SECONDARY HOOK (Slides 2-3)

**When:** Slides 2 and 3, each a standalone hook for cold scrollers.

**Prompt template:**
```
Instagram carousel slide, 1080x1350 portrait format. Warm [cream/beige/blush] background with paper grain texture, matching the editorial style.

[VISUAL ELEMENT - lighter than cover:]
- Small icon or illustration related to the slide topic, positioned center or center-right
- Hand-drawn annotation: [circle/arrow/underline] emphasizing one area
- Optional: small screenshot thumbnail in rounded card

Large clear negative space for bold headline text (center, upper half). Clean breathing room.

Same warm editorial aesthetic as previous slides. Consistent grain texture and color temperature.

DO NOT include any text, words, letters, numbers, or written content in the image.
```

---

### Type: CONTENT - STEPS (Numbered process)

**When:** Content explains a numbered how-to or process.

**Prompt template:**
```
Instagram carousel slide, 1080x1350 portrait format. Warm beige background (#EDE8DF) with paper grain.

[VISUAL ELEMENT:]
- [If screenshot needed:] Rounded card container (12px radius, subtle shadow) in center showing [TOOL/APP] interface screenshot, approximately 800x500px area
- [If code needed:] Dark terminal block (#1E1E2E) with three colored dots (red, yellow, green) in top-left corner, monospace code area, approximately 900x300px
- [If neither:] Small illustrative icon or hand-drawn diagram related to the step

Hand-drawn [circle/arrow] annotation pointing to the key area of the visual element.

Leave space for: step number and label (top-left area), headline (above visual element), body text (below visual element), small handle (bottom-right).

Warm editorial style, consistent with carousel series.

DO NOT include any text, words, letters, numbers, or written content in the image.
```

---

### Type: CONTENT - CHECKLIST (List of items)

**When:** Content is a list of tips, tools, or items.

**Prompt template:**
```
Instagram carousel slide, 1080x1350 portrait format. Warm cream background with paper grain.

Visual layout suggesting a checklist or list format:
- [NUMBER] rows of subtle horizontal divider lines or light card sections, evenly spaced vertically
- Hand-drawn checkmark symbols in [ACCENT COLOR] at the left margin of each row
- One row highlighted with a slightly darker background tint or accent color border
- Small hand-drawn arrow pointing to the highlighted row

Leave generous space in each row for text content. Leave top area clear for headline.

Warm editorial style, hand-crafted checklist feel, like a designer's notebook page.

DO NOT include any text, words, letters, numbers, or written content in the image.
```

### Type: CONTENT - SCREENSHOT (Tool demonstration)

**When:** Content demonstrates a specific tool, website, or app interface.

**Prompt template:**
```
Instagram carousel slide, 1080x1350 portrait format. Warm cream background (#F5F0E8) with paper grain.

Central element: Large rounded card container (12px radius, subtle drop shadow) taking up approximately 60% of the slide area (center). Inside the card: [TOOL] interface screenshot (provided as reference image).

Decorative elements:
- Hand-drawn curved arrow pointing to a key feature area of the screenshot
- Small sticky note / tape callout element near the arrow (yellow-cream, slightly rotated, with tape graphic at top)
- Subtle hand-drawn circle around one area of the screenshot

Leave top area clear for headline text. Leave bottom area clear for body text and handle.

Editorial tech-tutorial aesthetic. Warm and approachable, not corporate.

DO NOT include any text, words, letters, numbers, or written content in the image.
```

### Type: CONTENT - DATA (Stats, comparisons, charts)

**When:** Content presents data, statistics, or comparisons.

**Prompt template:**
```
Instagram carousel slide, 1080x1350 portrait format. Warm beige background with paper grain.

Central infographic element:
- [CHART TYPE: horizontal bar chart / donut chart / comparison table / stat callout boxes / flow diagram]
- Data visualization showing [DATA DESCRIPTION] with [NUMBER] data points
- Color scheme: warm neutrals with [ACCENT COLOR] for the primary/highlighted data point
- Rounded edges on all chart elements, hand-drawn quality feel
- Small source attribution area at bottom of the chart

[If comparison:] Two side-by-side rounded cards or columns

Hand-drawn annotation: arrow or circle highlighting the most important data point.

Leave top area clear for headline text. Leave bottom for attribution.

Infographic style that matches the warm editorial aesthetic. NOT a corporate dashboard. Hand-crafted data visualization feel.

DO NOT include any text, words, letters, numbers, or written content in the image.
```

**Alternative for complex data:** Use NotebookLM `studio_create` with `artifact_type=infographic`, then feed the result to Nano Banana as a reference image for re-styling.

### Type: CONTENT - CODE (Terminal/code display)

**When:** Content shows commands, code snippets, or terminal output.

**Prompt template:**
```
Instagram carousel slide, 1080x1350 portrait format. Warm cream background (#F5F0E8) with paper grain.

Central element: Dark code/terminal block (#1E1E2E) with:
- Three colored dots (red #FF5F56, yellow #FFBD2E, green #27C93F) in the top-left corner
- Monospace code lines inside (colored syntax: green for commands, white for text, gray for comments)
- Rounded corners (8px), subtle glow/shadow around the block
- Block size approximately 900x400px, centered

Decorative elements:
- Hand-drawn arrow pointing to the terminal block from above
- Small sticky note callout to the right of the terminal

Leave top area clear for headline text. Leave bottom clear for body text and handle.

Tech-editorial style. The terminal should feel like a screenshot from a developer's setup, embedded in a warm editorial layout.

DO NOT include any text, words, letters, numbers, or written content in the image.
```

---

### Type: CTA (Final slide)

**When:** Always the last slide.

**Prompt template:**
```
Instagram carousel slide, 1080x1350 portrait format. Warm cream background, slightly lighter and more open than content slides.

Minimal visual elements:
- Subtle [ACCENT COLOR] gradient or glow in the center area (soft, not sharp)
- Hand-drawn decorative elements: small stars, a curved arrow pointing down (toward where the handle will go)
- Optional: small avatar/profile image element in a circle frame, center-bottom area
- Overall feel: clean, spacious, inviting action

Large clear space in center for CTA headline text. Clear space below for handle.

The cleanest, most spacious slide in the carousel. Invites the viewer to take action.

DO NOT include any text, words, letters, numbers, or written content in the image.
```

## Art Direction Decision Tree

When analyzing each slide's content, match to a type:

1. Is it slide 1? → **COVER**
2. Is it slide 2 or 3? → **SECONDARY HOOK**
3. Is it the last slide? → **CTA**
4. Does it mention a specific tool, app, or website that can be screenshotted? → **SCREENSHOT**
5. Does it contain data, stats, percentages, or comparisons? → **DATA**
6. Does it contain code, commands, terminal output, or API calls? → **CODE**
7. Is it a numbered process or step? → **STEPS**
8. Is it a list of items, tips, or features? → **CHECKLIST**
9. None of the above? → **STEPS** (default, most versatile)

---

## Visual Consistency Rules

1. **Same conversation_id** for all slides in a carousel (Nano Banana session)
2. **First slide includes style references** from `references/` folder
3. **Subsequent slides** use `use_image_history: true`
4. **Same accent color** specified in every prompt for the carousel
5. **Same background base tone** in every prompt (pick one: cream, beige, or blush)
6. **Same grain texture description** in every prompt
7. **Annotation style consistent** across all slides (same "hand-drawn" descriptor)

## Anti-Patterns

NEVER generate slides that look like:
- Canva templates (gradients, icon libraries, stock photography, rounded sans-serif everything)
- "Bro marketer" (neon colors, all-caps Impact font, fake urgency, dollar signs)
- Corporate presentations (PowerPoint feel, bullet points, clip art)
- Pure minimalism (just text on flat white - needs texture, annotations, personality)
- Over-designed (too many elements, competing visuals, visual noise)
- Emojis as design elements (no emoji in the visual layer)
